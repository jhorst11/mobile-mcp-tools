# iOS Basic Requirements

The following software is required for all Mobile SDK development.

- A Salesforce [Developer Edition organization](dev-de.md) with a [connected app](connected-apps-howto.md).

iOS development with Mobile SDK 13.0 also requires the following software.

- iOS SDK:
  - Deployment target: iOS 17
  - Base SDK: iOS 18
- Xcode version: 16 or later. (We recommend the latest version.)

:::note

Mobile SDK for iOS supports Cocoa Touch dynamic frameworks.

:::
